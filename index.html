Análise de Churn Bancário
📓 Notebook 📊 Ver dashboard
Sobre o projeto

Identifica clientes com alta probabilidade de cancelamento antes que saiam, permitindo ação proativa de retenção. Conta com dashboard interativo em React para visualização dos resultados, com rastreamento automático via MLflow.

Adquirir um novo cliente custa 5–7× mais do que reter um existente. Este projeto detecta sinais de saída antes do cancelamento — probabilidade de churn calculada por perfil, com recomendações de ação para cada nível de risco, tudo acessível via dashboard em React. O desafio principal é o desequilíbrio nos dados — clientes que cancelam são sempre minoria. Para resolver isso, foi usado SMOTE combinado com rastreamento de experimentos via MLflow.

Resultado prático
📌 Impacto

Ação preventiva antes da saída do cliente. Prioriza esforços de retenção nos perfis de maior risco, com decisões consistentes e auditáveis. Bancos e fintechs acionam equipes de retenção antes do cancelamento. Serviços de assinatura reduzem cancelamentos com ofertas personalizadas. Times comerciais otimizam seu esforço focando nos perfis de maior risco.

Cliente com alto risco de churn (71,1%). Principais fatores: satisfação crítica (2/5), reclamação registrada não resolvida e apenas 2 anos no banco. Risco composto elevado mesmo sendo membro ativo. Ignorar esse perfil tende a resultar em cancelamento a curto prazo.

2Contato pessoal imediato — entender as razões da insatisfação e mostrar que o banco se importa com o cliente.
3Oferta personalizada — promoções e benefícios exclusivos baseados no perfil do cliente para aumentar o vínculo.
4Programa de fidelidade — recompensar a lealdade e incentivar a continuidade com o banco.
5Acompanhamento regular — monitorar a satisfação periodicamente para evitar que a insatisfação avance.

Sobre o projeto

Transforma milhares de avaliações de clientes em inteligência de produto automaticamente. Analisa sentimento, extrai palavras-chave e gera resumos executivos por produto usando NLP clássico e modelos de linguagem.

Empresas com catálogos de centenas de produtos recebem milhares de reviews diariamente — ler manualmente é inviável. Este pipeline transforma texto bruto em inteligência estruturada: sentimento por produto, tendências emergentes e resumos executivos gerados por LLM, prontos para decisão sem intervenção humana.

Resultado prático
📌 Impacto

Elimina análise manual de reviews. Entrega sentimento, tendências e recomendações prontas para decisão. E-commerce monitora reputação de produtos em tempo real. Times de produto identificam o que clientes valorizam e o que gera reclamação. Marketing extrai atributos associados a avaliações positivas para usar em campanhas.

Análise de Vendas — Dashboard & SQL
🗄️ SQL Analysis 📊 Ver dashboard
Sobre o projeto

Projeto dual de análise de vendas: um dashboard executivo interativo no Power BI e uma análise SQL completa em Python via SQLAlchemy e Pandas. Os dois lados se complementam — o dashboard entrega visão rápida para gestores, enquanto a análise SQL aprofunda os dados com queries analíticas e gráficos gerados diretamente do banco.

O componente SQL utiliza SQLAlchemy para conectar ao banco de dados e executar queries analíticas diretamente em Python. As análises incluem ranking de produtos e marcas, variação de receita por período, comparativo de faturamento vs. gastos e cálculo de margem — todas com visualizações geradas em Matplotlib e Seaborn, tornando os resultados prontos para apresentação sem depender do Power BI.

O dashboard Power BI consolida 247 mil transações em painel dinâmico filtrado por período, região e categoria — qualquer stakeholder extrai respostas sem depender de um analista, enquanto os notebooks SQL permitem análises customizadas e reproduzíveis para times técnicos.

Resultado prático
📌 Impacto

Visão executiva em tempo real e análises SQL reproduzíveis. Decisões de produto, região e mix de marcas com base em dados. Gestão comercial acompanha faturamento e margem sem relatórios manuais. Times técnicos executam queries customizadas e geram gráficos diretamente do banco. Alta gestão tem painel consolidado para decisões estratégicas com drill-down completo.

Aprovação de Crédito com ML
📓 Notebook 🚀 Testar o modelo
Sobre o projeto

Automatiza decisões de concessão de empréstimos bancários e calcula a taxa de juros ideal para cada perfil de cliente. Combina um classificador de aprovação com um modelo de regressão para precificação de risco.

Decisões de crédito mal calibradas custam bilhões anualmente em inadimplência ou em clientes bons que são negados injustamente. Este projeto substitui intuição por predição quantitativa — dois modelos em pipeline: um classificador para aprovar ou negar, e um regressor para calcular a taxa proporcional ao risco real de cada perfil.

Resultado prático
📌 Impacto

Reduz risco de inadimplência e acelera aprovações. Substitui intuição por decisão baseada em dados, com transparência total sobre os fatores de cada decisão. Bancos e fintechs automatizam análises que dependem de analistas humanos. Cooperativas oferecem taxas personalizadas e justas por perfil. Empresas de empréstimo escalam operações com decisões consistentes e auditáveis.
